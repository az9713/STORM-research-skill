# Source Basis

This package is reverse-engineered from the local transcript of Nate Herk's video:

- `Stanford's Method Turns Claude Into a PHD Level Research Team`
- `https://www.youtube.com/watch?v=Tj3018n5MVg&t=14s`
- Transcript: `video_sources/Tj3018n5MVg/transcript.clean.txt`

The transcript says the original workflow:

- Uses five lenses: practitioner, academic, skeptic, economist, historian.
- Starts with topic scoping.
- Maps contradictions between perspectives.
- Synthesizes into a self-contained HTML report.
- Performs adversarial peer review and citation verification.
- Labels sources as confirmed, corrected, or demoted.
- Uses a report template HTML file for consistent output.
- Can be customized with extra lenses such as beginner in AI or content creator.

The exact original prompts and template are not visible in the video. Files in this folder are a faithful reconstruction of the described behavior.
